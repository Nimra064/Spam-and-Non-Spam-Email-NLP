# 📧 Spam and Non-Spam Email Classification using NLP
---

## 📌 Project Overview

This project is a Natural Language Processing (NLP) based machine learning model that classifies emails as **Spam** or **Not Spam (Ham)**.
It uses text preprocessing techniques and supervised learning algorithms to detect unwanted emails and improve email filtering systems.

---

## 🎯 Objective

- Build an NLP pipeline for email text classification  
- Detect spam emails automatically  
- Apply text preprocessing and feature extraction techniques  
- Train machine learning models on real-world text data  

---

## ⚙️ Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/Nimra064/Spam-and-Non-Spam-Email-NLP.git

# 2. Navigate to the project directory
cd Spam-and-Non-Spam-Email-NLP

# 3. Install required dependencies
pip install -r requirements.txt

# 4. Launch Jupyter Notebook
jupyter notebook
````
